# Seed Mapping Documents

本目录保存早期整理得到的 Hack@DAC 描述级映射文档。

这些文档用于初始化后续 benchmark 数据集整理流程中的：

- `official_gt.csv`
- `task_gt.csv`
- `batch_plan.md`

使用限制：

- 这些文档是 description-level screening，不是 RTL-confirmed GT。
- 文档中的权限相关、跨模块、初始类别和纳入状态只能作为初筛参考。
- 文档中的 `v0.2 参考标签` 不作为最终特征定义或实验标签。
- 进入实验前必须继续完成 case-level GT、RTL evidence 复核和 evidence-level GT。
