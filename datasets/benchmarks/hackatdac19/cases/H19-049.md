# H19-049 Case GT

## 1. Case Identity

| 字段 | 内容 |
|---|---|
| `case_id` | H19-049 |

| `official_description` | JTAG password is only 32 bits |
| `task_category` | `JTAG/authentication_strength` |
| `gt_status` | `rtl_confirmed` |

## 2. Expected Bug Semantics

The JTAG password path uses a 32-bit key and 32-bit comparison value. `jtag_key` is declared as `[31:0]`, the internal `pass` register is 32 bits, and the DTM_PASS operation compares `data_d` against `pass`. This makes the JTAG password only 32 bits wide.

## 3. Expected RTL Evidence

| Evidence | 文件 | 行号 | 关键对象 | 评分用途 |
|---|---|---:|---|---|
| 32-bit JTAG key input | `third_party/hackatdac19/src/debug/dmi_jtag.sv` | 20-25 | `jtag_key[31:0]` | Shows the JTAG key input width is 32 bits. |
| 32-bit password storage | `third_party/hackatdac19/src/debug/dmi_jtag.sv` | 76-80 | `data_d`, `data_q`, `pass` | Shows the password/reference value is 32 bits. |
| Password comparison | `third_party/hackatdac19/src/debug/dmi_jtag.sv` | 110-117 | `DTM_PASS`, `data_d == pass` | Shows authentication compares only the 32-bit value. |
| Key loaded into password register | `third_party/hackatdac19/src/debug/dmi_jtag.sv` | 228-233 | `pass`, `jtag_key` | Shows the 32-bit key is used as the password reference. |

## 4. Hit Criteria

- Identifies that the JTAG password/key is only 32 bits.
- Cites the `jtag_key` and `pass` widths and the comparison.
- Explains this as an authentication-strength issue.

## 5. Partial Hit Criteria

- Reports generic JTAG weakness without naming the 32-bit width.
- Finds the hardcoded JTAG key but not the password width; hardcoding is H19-054.

## 6. False Positive / False Negative Rules

False positive examples:

- Claims JTAG instructions bypass password; that is H19-047.
- Claims JTAG key is hardcoded; that is H19-054.

False negative condition:

- If the input scope contains `dmi_jtag.sv`, but the agent does not report the 32-bit JTAG password width, mark H19-049 as false negative.

## 7. Evaluation Notes

This case is about password width, not password storage origin or reset behavior.
