## 2. Expected Bug Semantics

The CSR privilege check compares current privilege against the decoded CSR privilege level, but it contains a special exception for `CSR_MEPC`. This permits a CSR access path that bypasses the normal privilege check for `MEPC`, allowing lower-privilege access to a machine-level CSR.
