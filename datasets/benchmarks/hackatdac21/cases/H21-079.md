# H21-079 Case GT

## 1. Case Identity

| 字段 | 内容 |
|---|---|
| `case_id` | H21-079 |
| `official_id` | #79 |
| `official_description` | Some SHA registers are not locked by register locks |
| `task_category` | `sha/register_lock` |
| `gt_status` | `rtl_confirmed` |

## 2. Expected Bug Semantics

SHA data registers are protected inconsistently: writes to `data[0]` through `data[15]` are guarded by `reglk_ctrl_i[3]`, while reads of the same registers are guarded by `reglk_ctrl_i[2]`. A lock setting that protects one access direction does not consistently protect the same SHA registers.

## 3. Expected RTL Evidence

| Evidence | 文件 | 行号 | 关键对象 | 评分用途 |
|---|---|---:|---|---|
| Write lock path | `third_party/hackatdac21/piton/design/chip/tile/ariane/src/sha256/sha256_wrapper.sv` | 118-156 | `data[0]` ... `data[15]`, `reglk_ctrl_i[3]` | Shows SHA input writes use one lock bit. |
| Read lock path | `third_party/hackatdac21/piton/design/chip/tile/ariane/src/sha256/sha256_wrapper.sv` | 168-203 | `data[0]` ... `data[15]`, `reglk_ctrl_i[2]` | Shows reads of same registers use a different lock bit. |
| Peripheral integration | `third_party/hackatdac21/piton/design/chip/tile/ariane/openpiton/riscv_peripherals.sv` | 1443-1454 | `i_sha256_wrapper` | Shows SHA is wired into the peripheral subsystem. |

## 4. Hit Criteria

- Reports inconsistent register-lock mediation for SHA data registers.
- Cites write guard `reglk_ctrl_i[3]` and read guard `reglk_ctrl_i[2]`.
- Explains how this can leave SHA protected registers insufficiently locked.

## 5. Partial Hit Criteria

- Reports SHA state residue but not register-lock inconsistency.
- Finds one SHA lock bit but not the read/write mismatch.

## 6. False Positive / False Negative Rules

False positive examples:

- Claims SHA hash digest is secret without lock evidence.
- Reports H21-036 only.

False negative condition:

- If the input scope contains `sha256_wrapper.sv`, but the agent does not report inconsistent lock mediation over SHA data registers, mark H21-079 as false negative.

## 7. Evaluation Notes

This case is distinct from H21-036: H21-036 is state residue; H21-079 is lock mediation inconsistency.
