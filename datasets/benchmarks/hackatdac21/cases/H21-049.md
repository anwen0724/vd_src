## 2. Expected Bug Semantics

`acct_wrapper` computes `acc_ctrl_o` by ORing `acct_mem[3*0+0]` with `{8{we_flag}}`. The `we_flag` signal is an external control wired from the peripheral subsystem, so it can corrupt effective access-control output independent of stored access policy.

- Explains that effective policy can differ from stored `acct_mem`.

## 6. False Positive / False Negative Rules

- Reports register-lock corruption instead of access-control corruption.
