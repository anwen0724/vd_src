## 2. Expected Bug Semantics

The privilege-indexed access-control matrix computes `acc_ctrl_c[i][j]` with an extra condition for `j==5`. Since HMAC uses peripheral index 4 and PKT uses index 5, the expression lets HMAC authorization also enable PKT access.
